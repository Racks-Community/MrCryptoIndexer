# Mr. Crypto Indexer

## Get started

### Install dependencies

```bash
pnpm i
```

### Create `.env` file and complete it

```bash
cp .env.example .env
```

### Seed database

```bash
pnpm db:seed
```

### Build and run

```bash
pnpm build
pnpm start
```
